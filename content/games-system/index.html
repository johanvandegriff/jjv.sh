--- title: "The Glamorous Automatic Multi-game Emulator System (GAMES)" date: 2019-09-03T16:34:13-04:00 tags: [] ---

GAMES is a tiny Arduino game system with a 3x3 LED screen and 4 buttons. Version 1, which uses an Arduino Pro Mini, is done; and version 2, which will use an ATTiny microcontroller, is a work in progress.
